# Snake Game

A classic snake game with modern UI and smooth gameplay.

## How to Run

1. Open terminal in this folder
2. Run the server:
   ```bash
   node server.js
   ```
3. Open browser to: http://localhost:3006

## Controls

- **Arrow Keys** - Move snake
- **Hold key** - Speed boost (1.5x)

## Features

- Snake starts with 8 units
- Speed increases every 5 hearts collected
- Score and best score tracked
- Green border on playing box
