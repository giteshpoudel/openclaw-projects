# openclaw-projects

Collection of projects managed by OpenClaw AI agent squad.

## Projects

### Snake Game
- **Description:** Classic snake game with modern UI
- **Features:** 
  - Snake starts with 8 units
  - Speed increases every 5 hearts collected
  - Score and best score displayed outside game
  - Green border on playing box
- **Tech:** HTML5 Canvas, Vanilla JavaScript
- **Status:** Active development

## Agent Squad

This repo is managed by a team of 4 AI agents:

| Agent | Role | Responsibilities |
|-------|------|------------------|
| **PM** | Project Manager | Creates PDR, coordinates tasks, tracks progress |
| **BA** | Business Analyst | Writes requirements, defines specs |
| **DEV** | Developer | Implements features, writes code |
| **TESTER** | Quality Assurance | Tests features, reports bugs |

### Workflow
1. **You** request a feature/fix
2. **PM** creates project plan and PDR
3. **BA** writes detailed requirements
4. **DEV** implements the code
5. **TESTER** verifies the work
6. You review in Cursor and push to GitHub when ready

All activity is tracked via the local dashboard.
