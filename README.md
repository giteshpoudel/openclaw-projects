# openclaw-projects

Collection of projects managed by OpenClaw AI agent squad.

## Projects

### Snake Game
- **Description:** Classic snake game with modern UI
- **Features:** 
  - Snake starts with 8 units
  - Speed increases every 5 hearts collected
  - Score and best score displayed outside game
  - Green border on playing box
- **Tech:** HTML5 Canvas, Vanilla JavaScript
- **Status:** Active development

## Adding New Projects

1. Create a new folder in root
2. Add a `server.js` for local hosting
3. Update README.md with project details

## Local Development

Each project has its own server. Run with:
```bash
node server.js
```
